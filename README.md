# iPhone Dynamic Wallpaper Service (R2 + Workers)

這是一個基於 Cloudflare Workers 和 R2 建立的動態桌布 API，專為 iPhone 自動化設計。

## 🚀 API 說明

### 1. 隨機獲取桌布
- **URL**: `GET /random?folder=資料夾名稱`
- **範例**: `https://your-worker.me/random?folder=Work`
- **說明**: 從指定資料夾中隨機挑選一張圖片並回傳。

### 2. 上傳圖片
- **URL**: `POST /upload?folder=資料夾名稱`
- **Header**: `Authorization: Bearer YOUR_SECRET`
- **Body**: 圖片二進位資料
- **說明**: 讓 iPhone 捷徑直接將照片傳到 R2 儲存。

## 📱 iPhone 捷徑設定教學

要實現「自動換圖」與「分資料夾管理」，請依照以下步驟設定：

### 第一步：建立核心「換桌布」捷徑
1. 開啟 **捷徑 App**，建立新捷徑並命名為 `隨機換桌布`。
2. 點擊「新增動作」，搜尋 **「文字」**，內容輸入 `default` (這是預設資料夾)。
3. 在文字動作下方，點擊圖示選擇 **「接收輸入」** (Receive Input)，設為「捷徑輸入」。
4. 新增動作 **「取得網址內容」**：
   - 網址填入：`https://你的Worker網址/random?folder=`。
   - 在網址末尾點擊變數，選擇剛才的 **「文字」** 或是 **「捷徑輸入」**。
5. 新增動作 **「設定背景圖片」**：
   - 影像選擇剛才的「URL 內容」。
   - 點開「顯示更多」，**關閉「顯示預覽」**。

### 第二步：綁定「專注模式」自動化
1. 進入「設定」>「專注模式」，建立兩個模式：`工作` 與 `休息`。
2. 回到 **捷徑 App** > **「自動化」** 分頁。
3. 點擊 `+` 建立自動化，選擇 **「當『工作』模式開啟時」**：
   - 執行動作：選擇「執行捷徑」-> `隨機換桌布`。
   - 傳入參數：手動輸入 `Work`。
4. 同理，建立 **「當『休息』模式開啟時」**：
   - 執行動作：選擇「執行捷徑」-> `隨機換桌布`。
   - 傳入參數：手動輸入 `Life`。

### 第三步：快速上傳圖片 (分享選單)
1. 建立一個名為 `傳送到 R2` 的捷徑。
2. 設定「在分享表單中顯示」，輸入類型選「影像」。
3. 新增動作 **「取得網址內容」**：
   - 網址：`https://你的Worker網址/upload?folder=Work` (可依需求改)。
   - 方法：**POST**。
   - 標頭 (Headers)：`Authorization` 填入 `Bearer 你的金鑰`。
   - 要求內文：**檔案**，選擇該影像。

## 🛠️ 部署方式
1. 在 Cloudflare 控制台建立 R2 Bucket 名為 `my-wallpapers`。
2. 串接此 GitHub Repo 至 Cloudflare Workers。
3. 在 Workers 設定中新增環境變數 `AUTH_SECRET` 作為上傳金鑰。
4. **手動上傳第一張圖**：你可以透過 Cloudflare 網頁介面先手動建立資料夾並上傳一張圖，避免 API 回傳 404。
