# iPhone Dynamic Wallpaper Service (R2 + Workers)

這是一個基於 Cloudflare Workers 和 R2 建立的動態桌布 API，專為 iPhone 自動化設計。本手冊包含從零開始的完整部署與設定細節。

---

## 🛠️ 第一階段：Cloudflare R2 儲存空間設定
R2 是存放圖片的地方，完全私有且安全。

1.  **建立 Bucket**：
    *   登入 Cloudflare > 點左側 **R2** > **Create bucket**。
    *   命名為 `my-wallpapers`（需與 `wrangler.toml` 中的 `bucket_name` 一致）。
2.  **建立初始資料夾**：
    *   進入 `my-wallpapers` > 點 **Objects** > **Create folder**。
    *   建立一個名為 `default` 的資料夾。
    *   **手動上傳第一張圖**：上傳一張 9:16 的圖片到 `default` 裡，避免 API 第一次執行時回傳 404。
3.  **安全性**：維持預設的私有狀態，不需要開啟 Public Access。

---

## 🚀 第二階段：部署 Worker 與安全設定
1.  **環境變數 (AUTH_SECRET)**：
    *   進入 Worker 頁面 > **Settings** > **Variables**。
    *   點 **Add variable**：
        *   **Name**: `AUTH_SECRET`
        *   **Value**: 自己設定一組複雜的密鑰（例如 `my_ios_secret_123`）。這將用於上傳時的身份驗證。
    *   點 **Save and deploy**。
2.  **R2 綁定**：
    *   進入 **Settings** > **Bindings** > **R2 Bucket Bindings**。
    *   確保變數名稱是 `MY_BUCKET` 且指向您建立的 `my-wallpapers`。

## 📱 第三階段：iPhone 捷徑設定細節

### 1. 核心「隨機換桌布」捷徑 (抓圖)
1. 開啟 **捷徑 App**，建立新捷徑並命名為 `隨機換桌布`。
2. 新增動作 **「取得網址內容」**：
   - 網址：`https://你的Worker網址/random?folder=default` (資料夾可自訂)。
   - 方法：**GET**。
3. 新增動作 **「設定背景圖片」**：
   - 影像選擇剛才的「URL 內容」。
   - **關鍵細節**：點開動作中的「顯示更多」，**務必關閉「顯示預覽」 (Show Preview)**，否則自動化時會跳出確認視窗。

### 2. 綁定「專注模式」自動化
1. 進入 **捷徑 App** > **「自動化」** 分頁。
2. 點擊 `+` 建立自動化，選擇 **「專注模式」** (例如：工作)。
3. 選擇 **「開啟時」** 並設定為 **「立即執行」 (Run Immediately)**。
4. 執行動作：選擇「執行捷徑」-> `隨機換桌布`。

### 3. 快速上傳圖片 (分享選單)
1. 建立捷徑 `傳送到 R2`，在設定中開啟 **「在分享表單中顯示」**，類型選「影像」。
2. 新增動作 **「取得網址內容」**：
   - 網址：`https://你的Worker網址/upload?folder=default`。
   - 方法：**POST**。
   - 標頭 (Headers)：新增 `Authorization`，值填入 `Bearer 你的AUTH_SECRET`。
   - 要求內文：選擇 **「檔案」**，並指定為捷徑輸入的影像。

## 📊 API 說明
- **GET `/random?folder=xxx`**：隨機獲取桌布。
- **POST `/upload?folder=xxx`**：上傳圖片（需 Auth Header）。
- **GET `/folders`**：列出目前所有資料夾。
